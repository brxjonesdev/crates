Crates

Crates is a web app where users can create, share, and explore mixtapes around any theme, mood, or topic.

Core Features:

Create Crates: Build mixtapes by adding songs with basic metadata (title, artist, album).

Collaborate: Other users can suggest tracks to your Crates.

Save & Collect: Users can save Crates they like into their personal collection.

Search & Discover: Find Crates or tracks by title, artist, or tags.
